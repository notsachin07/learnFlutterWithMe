# Class 02 Exercises

Create `solution_02.dart` to solve these.

## Exercise 1: FizzBuzz (Classic Interview Question)
Write a loop that prints numbers from 1 to 20.
- If the number is divisible by 3, print "Fizz".
- If the number is divisible by 5, print "Buzz".
- If divisible by both 3 and 5, print "FizzBuzz".
- Otherwise, print the number.

## Exercise 2: Odd or Even
Write a program that defines an integer variable `number`.
- Use an `if/else` block to print "Even" if the number is even and "Odd" if it is odd.
- *Hint: Use the modulo operator `%`.*
