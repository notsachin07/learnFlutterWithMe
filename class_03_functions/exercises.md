# Class 03 Exercises

Create `solution_03.dart`.

## Exercise 1: The Converter
Write a function called `toFahrenheit` that takes a `double` (Celsius) and returns the Fahrenheit value.
* Formula: `(celsius * 9/5) + 32`
* Try to write this using **Arrow Syntax**.

## Exercise 2: The ID Card (Named Parameters)
Write a function called `printIDCard`.
1. It should use **Named Parameters**.
2. It requires `name` (String) and `age` (int).
3. It has an optional parameter `role` (String) with a default value of `"Student"`.
4. Call the function 3 times with different combinations.

*Hint for default value:* `String role = "Student"` inside the `{}`.

## Exercise 3: Calculator Function
Write a function `calculate` that accepts:
1. Two numbers (`a` and `b`).
2. A String `operation` (e.g., "add", "subtract").
3. Return the result based on the operation.
